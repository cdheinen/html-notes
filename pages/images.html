Images

Images can be displayed in HTML using the 'img' element. Note that this is a void element, meaning it does not need a closing tag because it is naturally empty and only defined using the element attributes.

The image that will be displayed is defined within the 'src' attribute of the 'img' element. Similar to the 'href' attribute of an anchor element, the 'src' attribute specifies the location of the image to be displayed, which can be both an absolute link or a relative link.

Alt Attribute

The 'alt' attribute of an 'img' element is used to provide alternate text for an image. This text will be displayed in place of an image if the image is unable to be loaded. It is also used for accessibility to describe the image to visually impared users.
