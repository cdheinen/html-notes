Example Link: About The Odin Project

Target Attribute

The above link opens The Odin Project About page in the current tab of the browser. To open the link in a new tab, we need to add the 'target' attribute. If the 'target' attribute is not specified, it will take on the '_self' value by default. Which will open the link in the current tab. To open the link in a new tab (or window), set the 'target' attribute to '_blank'.

Example Link (that will open in new tab): About The Odin Project

Rel Attribute

The 'rel' attribute is used to describe the relationship between the current page and the linked document. The 'noopener' attribute ensures that a link opened in a new tab or window cannot interact with or access the original page. Without it, the new page can use JavaScript to manipulate the original page, which is a security risk.

The 'rel' attribute can also be set to 'noreferrer', which will prevent new pages from knowing where the user came from by hiding the referrer, in addition to implementing the behavior provided by 'noopener'.

Types of Links

Absolute Links

An absolute link is a link to pages on other websites on the internet. An absolute link typically takes the form 'schema://domain/path'.

Relative Links

A relative link is a link to pages on our own website. Relative links do not include a domain because it assumes that the domain is the same as the current page. Here is an example relative link that will take you back to the homepage: Homepage.
