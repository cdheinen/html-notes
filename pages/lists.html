Lists

Unordered Lists

Unordered lists can be created using the 'ul' tag. As the name suggests, there is no visual ordering applied to each item in the list. Items can be added to the unordered list by adding a child 'li' tags under the parent 'ul' tag.

Here is an example of an unordered list:

  • Item 1
  • Item 2
  • Item 3

Ordered Lists

Ordered lists can be created using the 'ol' tag. As the name suggests, there is visual ordering applied to each item in the list. Items can be added to the ordered list by adding a child 'li' tag under the parent 'ol' tag.

Here is an example of an ordered list:

  1. Item 1
  2. Item 2
  3. Item 3
